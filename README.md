seo-auto-checker
================

#  Requriement
- pip install Scrapy
- python (i use version of 2.7.5)

# Command
- under the path of seo, just run:
    scrapy crawl seo -L WARNING
- you will see the output:
    equals.

# Extension
- just update the demo.csv with your url address and expected content of title, h1, description
